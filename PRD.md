# 프로젝트 기획서: 로컬 이주 시뮬레이션 플랫폼

## 🛠️ 기술 스택
* **Frontend:** React (모바일 웹 기준, 375px 너비 중심)
* **지도:** Kakao Map API
* **애니메이션:** Framer Motion
* **스타일:** Tailwind CSS

---

## 📐 전체 구조
* **탭 2개** (하단 고정 네비게이션)
* **온보딩** (최초 1회 — 라이프스타일 진단)

---

## 🚀 상세 흐름 및 기능

### 1. 온보딩 (최초 진입 시)
* **현재 사는 지역 선택** (ex. 서울)
* **라이프스타일 유형 진단** (4지선다 3-4문항)
* **결과:** 레저형 / 자연탐험형 / 디지털노마드형 / 집돌이형
* **결과 화면:** 유형 + 추천 지역 2-3개 미리보기
* **완료 후** 탭1 홈으로 진입

### 2. 탭1) 홈/떠나기 [행동 시작]
* **홈 화면 (기본)**
  * **중앙:** 봇짐 멘 캐릭터 (SVG) + 말풍선 *"오늘도 어디론가 떠나볼까?"*
  * **하단:** 현재 본 지역 표시 (ex. 📍 서울)
  * **하단 CTA 버튼:** "떠나기 🎒"
* **떠나기 화면**
  * **상단 텍스트:** "어디로 가볼까?"
  * 한반도 지도 (Kakao Map) 위에 추천 레지던스 마커 2-3개
  * **마커 디자인:** 집 모양 아이콘 + 지역명
  * **마커 누르면:** 레지던스 이름 / 프로그램 기간 / 라이프스타일 유형 매칭 이유 툴팁 표시
  * "여기로 떠나기" 버튼 누르면 → 이동 애니메이션 시작
* **이동 애니메이션**
  * 현재 본 지역 → 선택한 지역으로 화살표 그려지며 캐릭터 이동
  * 페이드 아웃 (화면 어두워짐)
  * 페이드 인 → 해당 지역 도착 화면
* **지역 도착 화면**
  * **상단:** *"📍 거제 도착!"* + 캐릭터가 그 지역에 서 있는 모습
  * **우편함 아이콘:** 클릭 시 귀촌 사례 / 주민 이야기 카드 1장 열람 가능
  * **미션 리스트**
* **미션 화면**
  * **일반 미션:**
    * 생활비 시뮬레이션 (월 예상 지출 입력 → 해당 지역 평균과 비교)
    * 교통 접근성 확인 (현재 본 지역 → 해당 지역 소요 시간)
    * 지역 통계 탐색 (인구 / 의료 / 마트 접근성 등)
    * 귀촌 사례 읽기
    * 전통시장 물가 체험 
    * 병원 접근성 확인
    * 하루 루틴 체험 (텍스트 기록)
    * 이주자 만나기 (텍스트 기록)
  * **최종 미션:**
    * 이주 결정 리포트 생성 (모든 미션 완료 후 활성화)
* **미션 수행 UX**
  
- 지도 위 미션 포인트로 이동 (카카오 로드뷰)
- 장소 도착 시 "들어가기" 버튼 → 실내 화면 전환
- 실내: 장소별 배경 사진 + 주민 3D 일러스트 캐릭터 등장
- 말해보카 스타일 대화 UI (말풍선 + 텍스트/선택지 답하기)
- 대화 완료 → 미션 클리어 + 축적 점수 업데이트

* **장소별 인터랙션**

- 가게/시장/병원: 실내 사진 + 캐릭터 대화형
- 바다/야외: 로드뷰 유지 + 사진 찍기 미션
- 
  * **각 미션 카드:** 제목 / 난이도 / 예상 소요 시간 / 완료 여부
  * **완료 시:** 축적 점수 +N / 적합도 점수 업데이트 애니메이션
  * **중간 이탈 시** 진행상황 자동 저장
  * **뒤로가기:** 미션 리스트 → 지역 도착 화면 → 한반도 지도 순으로

### 3. 탭2) 나의 여정 [축적&성장]
* **메인 화면**
  * 한반도 아트지도 전체 화면으로 펼쳐짐
  * 다녀온 지역마다 마커 표시
  * 축적 점수 높을수록 마커 크기 크게 / 색 진하게
  * **상단 고정 카드:** 가장 많이 탐색한 지역 1개 (축적 점수 기준)
  * **우측 상단 레이어 토글:** 축적 점수 보기 on/off / 적합도 점수 보기 on/off
* **지역 마커 클릭 시 바텀시트**
  * 지역명 + 방문 횟수
  * 축적 점수 (막대 그래프)
  * 적합도 점수 (막대 그래프)
  * 완료 미션 리스트
  * **CTA:** "이주 리포트 보기" (최종 미션 완료 시 활성화)
* **이주 리포트 화면**
  * 맞는 이유 vs 우려 이유
  * 핵심 지표 4개 요약
  * 지역 비교 (탐색한 지역들 나란히)
  * 공유 버튼
