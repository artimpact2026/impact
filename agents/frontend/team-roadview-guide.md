> 안녕하세요! 청풍 미션 화면에 들어갈 **로드뷰 자료**를 모아주시면 됩니다.
> 이 문서만 보고 작업 가능하도록 단계별로 정리했어요.

## 한눈에 보기

미션 하나당 두 가지를 모읍니다.

| 자료 | 무엇 | 용도 |
|---|---|---|
| 📷 **사진** | 미션 한 개당 **4~7장** | 사용자가 화살표 누르며 한 장씩 걸어가는 "미니 로드뷰" |
| 🔗 **네이버맵 공유링크** | 미션 한 개당 **1개** | 도착했을 때 "실제 로드뷰로 확인해보기" 버튼이 새 탭으로 여는 진짜 로드뷰 |

> 💡 **꼭 6장일 필요 없어요.** 짧고 단순한 코스면 4~5장이면 충분하고, 골목이 많이 꺾이는 코스는 7장까지 가도 됩니다. **실제 동선이 자연스럽게 이어지는 게 더 중요**합니다.

## 📷 자료 1: 사진 (4~7장)

### 어떤 컷을 잡나

미션 출발지 → 도착지까지 **실제 걸어가는 동선**을 따라가며, 아래 종류의 컷들을 섞어서 4~7장 잡으면 돼요.

| 컷 종류 | 언제 잡나 | 예시 |
|---|---|---|
| **출발** | 미션 시작점 (보통 레지던스/숙소 앞) | 순무민박 골목 입구 |
| **직선 구간** | 길이 한참 이어지는 곳 | 좁은 골목 한복판 |
| **회전 직전** | 갈림길·모퉁이가 보이는 시점 | 갈림길 안내가 보이는 곳 |
| **회전 직후** | 방향 튼 뒤 첫 컷 | 우회전 후 새 길의 첫 풍경 |
| **다가옴** | 도착지가 멀리 시야에 들어옴 | 시장 간판이 멀리 보임 |
| **도착** | 도착지 정면 (마지막 컷) | 시장 안 |


### 코스별 사진 개수 가이드 (참고용)

| 코스 성격 | 추천 컷 수 | 구성 |
|---|---|---|
| 짧고 직진만 있는 코스 | **4~5장** | 출발 / 직선 1~2 / 다가옴 / 도착 |
| 회전 1번 있는 코스 | **5~6장** | 출발 / 직선 / 회전 직전 / 회전 직후 / 다가옴 / 도착 |
| 회전 2번 이상 코스 | **6~7장** | 출발 / 회전 / 회전 / 직선 / 다가옴 / 도착 |

> **숫자에 너무 얽매이지 말고**, 실제 동선이 자연스럽게 따라가는 흐름이 만들어졌는지를 보세요. 무리해서 6장 채우려고 똑같아 보이는 두 컷 넣는 건 피해주세요.

### 캡처 방법 — 네이버지도 (PC 권장)

PC에서 작업하는 게 가장 깔끔합니다 (모바일도 가능하지만 UI가 더 많이 잘림).

1. https://map.naver.com 접속
2. 미션 도착지 검색 (예: "강화풍물시장")
3. 우측 컨트롤에서 **거리뷰** 아이콘 클릭 → 지도에 거리뷰 촬영 위치가 파란 선으로 표시됨
4. 출발지점(레지던스 앞) 위치를 클릭하면 거리뷰 진입
5. 화살표·드래그로 적절한 각도 잡기 → **스크린샷** (Mac: Cmd+Shift+4, Windows: Win+Shift+S)
6. **다음 지점으로 이동** — 거리뷰 화면의 진행 방향 화살표를 클릭하거나, 다음 위치를 클릭
7. 반복

#### 스크린샷 후 다듬기
- **네이버 워터마크·하단 UI를 잘라내주세요** (이미지 편집기로 크롭)
- 가로 1200px 이상 (모바일 풀스크린에 들어가서 작으면 흐려보임)
- 비율 4:3 또는 16:9 ** 근데 이거 읽고, 세로가 더 나을 거 같으면 말씀해주세요!! 저도 좀 고민됩니다 **

### 파일명 규칙

```
{미션ID}-{번호}.jpg
```

예시 (강화풍물시장 미션, 6장):
```

ganghwa-market-2.jpg   ← 2번 (우회전 직후)
ganghwa-market-3.jpg   ← 3번 (골목 갈림길)
ganghwa-market-4.jpg   ← 4번 (좌회전 직후)
ganghwa-market-5.jpg   ← 5번 (시장 간판 보임)
ganghwa-market-6.jpg   ← 6번 (도착: 시장 안)
```

### 사진과 함께 정리할 메모

사진과 같이 다음 표를 작성해서 보내주세요 (스프레드시트 한 장이면 충분).

| step | photo_file | 위치 라벨 (caption) | 다음 방향 (forward_direction) | 주민 멘트 (story, 선택) |
|---|---|---|---|---|
| 1 | ganghwa-market-1.jpg | 순무민박 앞 골목 | right | 민박 골목 입구에서 출발해요. 우측 길로 빠지면 읍내예요. |
| 2 | ganghwa-market-2.jpg | 우회전 후 읍내 진입 | straight | |
| 3 | ganghwa-market-3.jpg | 읍내 골목 갈림길 | left | 강화는 옛 도읍지라 길이 좁고 휘어요 |
| 4 | ganghwa-market-4.jpg | 좌회전 후 시장 방향 | straight | |
| 5 | ganghwa-market-5.jpg | 시장 간판이 보여요 | straight | |
| 6 | ganghwa-market-6.jpg | 풍물시장 안 (도착) | — | |

**컬럼 설명**:
- **위치 라벨**: 사용자에게 화면 상단에 보일 짧은 안내. 10자 안팎.
- **다음 방향**: 다음 사진으로 가려면 어디로 회전하는지. `straight` / `right` / `left` 중 하나. 마지막 사진(도착)은 비워둠.
- **주민 멘트**: 이 지점에서 자동으로 떠오를 NPC 한 줄. **모든 지점에 다는 건 산만하니까 출발지 + 인상적인 한두 지점에만** 추천. 비워도 OK.

## 🔗 자료 2: 네이버맵 도착 로드뷰 공유링크

### 무엇

사용자가 도착 화면에서 **"🗺️ 실제 로드뷰로 확인해보기"** 버튼을 누르면 새 탭으로 열리는 진짜 네이버 로드뷰 링크입니다. 미션 한 개당 1개 필요해요.

### 어떻게 따는지

#### 네이버지도 모바일 앱 (가장 쉬움)
1. 네이버지도 앱 열기
2. 도착지 검색 (예: "강화풍물시장")
3. **거리뷰** 아이콘 탭 → 도착지 정면이 잘 보이는 지점 선택
4. 손가락으로 화면 회전·확대해서 **사용자에게 보여주고 싶은 첫 화면**을 잡기
5. 우상단 **공유** 버튼 → **주소 복사**
6. `https://naver.me/XXXXXXXX` 형식의 짧은 링크가 복사됨

#### 네이버지도 PC
1. https://map.naver.com 에서 도착지 검색
2. 거리뷰 진입 → 적절한 위치·각도 잡기
3. 우상단 **공유** 아이콘 → **URL 복사**
4. (PC URL이 길어요 — 모바일에서 단축링크 받는 게 더 깔끔합니다)

### 카카오맵도 OK (선택사항)

네이버에 거리뷰가 없는 지역(드뭄)이거나 카카오가 더 디테일이 좋은 경우엔 카카오맵 링크도 됩니다. 형식: `https://kko.to/XXXXXXXX`

## ❓ 자주 묻는 질문

**Q. 5장보다 적게 잡아도 되나요?**
A. 네, **4장도 OK**입니다. 단 출발-중간-다가옴-도착 흐름은 보여야 해요. 너무 짧으면 사용자가 "어, 벌써 도착?" 하고 몰입이 깨질 수 있어요.

**Q. 7장 넘어도 되나요?**
A. 가능은 하지만 사용자가 화살표를 7번 넘게 누르면 지칠 수 있어요. **8장 이상은 추천 안 함**.

**Q. 회전이 없는 일직선 코스인데도 5장 채워야 하나요?**
A. 아니요. 직선만 있는 코스는 4~5장이 자연스럽고, 더 채우려고 비슷한 두 컷 넣으면 오히려 단조로워져요.

**Q. 도착지가 너무 가까워서 사진이 2~3장으로도 충분할 것 같은데?**
A. 그런 경우는 미션 자체가 "도보 이동" 컨셉이 안 어울리는 거예요. 그 미션은 일단 **건너뛰고** 개발팀에 알려주세요 (모드를 바꾸거나 미션 위치를 조정할 수 있어요).

**Q. 네이버에 거리뷰가 없는 지역이에요.**
A. 카카오맵에 있는지 확인해주세요. 둘 다 없으면 **현장 사진**으로 대체 가능합니다 (직접 촬영 또는 인터넷 검색). 개발팀에 미리 알려주세요.

**Q. 사진 화질이 별로면 캡처 다시 해야 하나요?**
A. 큰 차이 없으면 그대로 OK. 단 너무 흐릿하거나 워터마크가 가운데 박혀있으면 다시 부탁드려요.

**Q. `주민 멘트`(story)는 누가 쓰나요?**
A. 작업해주시는 분이 캡처 풍경에 어울리는 한 줄 정도 적어주시면 좋습니다. 어렵다면 비워두고 개발팀이 채워 넣을게요.
