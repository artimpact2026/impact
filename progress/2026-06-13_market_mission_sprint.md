## 1. 강화풍물시장 미션 — 1층 좌판 → 2층 비빔 (가장 큰 작업)

### 흐름 전체
```
미션 정보 화면
   ↓ 체험하기
카카오 로드뷰 (후문 도착지점, 60m 안 자동 도착)
   ↓
인트로 스플래시 3장 (정문 → 지도 → 1층)
   ↓
1층 좌판 (6품목 — 카드 톤 회색↔컬러 + 드래그 + 흥정 + 7,500↑ 시 바람이 안내)
   ↓ "2층 올라가서 밴댕이 한 입 하기"
계단 트랜지션 (1.1s in + 1.5s hold)
   ↓
2층 인트로 스플래시 ("와~ 2층은 전부 밴댕이네!")
   ↓
회무침 비비기 + 회무침/밴댕이 한 점씩 먹기
   ↓ "잘 먹었어요 — 한 줄 기록 남기기"
미션 완료 결과 카드
   ↓ "확인"
집 화면 (residence-home) 직행
```

### 1층 좌판
- 6품목 (밴댕이 제거, 생순무 추가): 새우젓 8k / 순무김치 5k / 약쑥떡 3k / 고구마말랭이 2.5k / 국화빵 2k / 생순무 2k
- 예산 10,000원 (빠듯 — 골라야 함)
- **카드 톤 회색 ↔ 컬러** — 탭하면 활성화, 길게 잡고 드래그하면 장바구니로
- 흥정 모달: "깎아주세요" / "그냥 살게요" / 취소 + 품목별 성공률(0.45~0.8)
- **실제 시세 fact** 반영 ("전국 젓새우 70%가 강화" / "순무김치 3kg 33,000원" 등)
- 사장님은 흥정 모달에서만 등장 (`clay-jieum-solo`)
- **장바구니 인라인 SVG** (`basket-bag`, 라탄 가방) — 이전 png 자산 폐기
- spent ≥ **7,500원 도달 시 바람이 floating** 등장 → 탭하면 2층

### 2층 비빔
- **인터랙션 변천사** (이번 스프린트 3번 다시 만듦):
  1. 색칠(brush canvas) — 별로 → 폐기
  2. 입자 물리 시스템 — 별로 → 폐기
  3. **손가락 자취 양념 트레일** (최종) — DOM 직접 조작, 0.55~0.95 alpha
- 그릇은 **둥근 한식 밥그릇 SVG** (정면 반원 사발 + 윗단 림)
- 비비기 진행도 따라 **saturate 0.85 → 1.35 실시간 변화** + 그릇 wobble
- 실시간 사장님 멘트 (6단계, 진행도 따라 자동 전환)
- 비빈 후 회 5점 spring 등장
- 사이드: **밥공기** (시각 데코) + **밴댕이회 접시** (탭으로 6점)
- 회무침 5점 + 밴댕이 6점 = 총 11번 탭으로 완료

### 카피
- title: "강화풍물시장 둘러보기" (짧게, 스포 없이)
- description: "강화 사람들이 가장 자주 찾는 시장 한 바퀴"
- curiosity: "오늘 시장에서 만나는 강화 한 상은 어떤 모습일까?"

## 2. 카카오 로드뷰 정확도 + 시인성 개선

### 화살표 방향
- 카카오 SDK `viewpoint.pan` 단위 (0~360) vs 우리 `bearingDeg` (-180~180) 충돌 → **`normalize360()` 로 통일**
- `relative = (bearing - viewHeading + 360) % 360`
- 화살표 회전이 사용자 시야 기준 실제 도착지 방향 정확히 가리킴

### 4방향 라벨 (시인성)
- `relative` 따라:
  - ⬆ 그대로 직진 (±45°)
  - ➡ 오른쪽 골목으로 (45~135°)
  - 🔄 뒤로 돌아가기 (135~225°)
  - ⬅ 왼쪽 골목으로 (225~315°)
- 큰 글자 라벨 + 회전 화살표 + 거리 카드 한 줄

### 위치 변경
- 화살표 카드: 정중앙 → **상단 (top-[68px])** (사용자 요청)
- "🔍 오늘의 미션" 호기심 카드 통째 제거

### 도착 처리
- 임계 30m → **60m** (정문 panoId 미등록 대응)
- 도착(`arrived`) 후에만 CTA "정문으로 들어가기" 노출 (이전엔 도착 전에도 떠 있던 어색함)

### travelGuide 두 단계
- 평소 (≥100m): "강화 골목 한 번 둘러보면서 가봐요"
- 도착 직전 (<100m): "이제 다 왔어요. 곧 시장 후문이 보일 거예요!" (`travelGuideArrival` 새 필드)
- `destinationLabel: "강화풍물시장 후문"`

## 3. BottomNav 게이팅 + `--content-bottom` 동적

### 게이팅 변경 (사용자 피드백 2번 거침)
- 1차: 청년마을 진입 후 다 숨김 → 너무 광범위
- 최종: **미션 진입(`mission-list` ~ `mission-complete`) 만 숨김**
- 본가/이동/도착/집/의식/퍼즐/우편함은 모두 BottomNav 보임

### CSS 변수 동적
- `hideNav` 토글 시 `document.documentElement.style.setProperty('--content-bottom', '0px')`
- 미션 화면이 BottomNav 자리(~96px)까지 풀스크린 활용
- 다시 본가 돌아가면 `removeProperty` 로 원래 값 복귀

### 집에 본가 복귀 CTA
- `ResidenceHomeScreen` 우상단 `🏠 본가로 돌아가기` 칩
- `onReturnHome` 호출 → `traveling-back`
